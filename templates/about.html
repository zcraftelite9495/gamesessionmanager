About This Program

Z's Game Session Manager

Backstory

Back in May of 2024, I created a Google Sheet to show my friends how good I was at organizing data. What did I do with this sheet, you ask? I used it to track the data of every single game I play on every single platform. And I did it all manually. It was [more than] a little bit time consuming and involved me taking a lot of screenshots on my phone when I started and stopped playing a game. So, back in September of 2024, I started working on a Terminal app I could use to put in the data more easily. And that was v1.0.0. Eventually, I got bored in life for quite a bit so I decided to expand on it, and eventually, it became what it is today, version {{ gsmversion }} out for everyone to see. I published the code on GitHub to show off my code, and I actually use this project daily instead of manually logging everything, and it makes it so much more easier.

What is this for?

This program integrates with a Google Sheet I designed in order to track, display, and summarize the playtime stats for all my games and for all the platforms I play on. I published the code because people on GitHub might find my project interesting [I hope] and I might be able to use it on my resume if my Science career ever falls through and I become a full time programmer.
